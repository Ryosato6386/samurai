# gitについて

## 主に使用するコマンド

**git clone**<br>
リモートリポジトリのデータをローカル環境に持ってくる

**git add .**<br>
編集内容を追加

**git commit -m"coment"**<br>
addで追加したものを保存する

**git push URL**<br>
commitでローカルに保存したものをリモートリポジトリに保存する

**git pull URL**<br>
mergeとフォルスを同時に行う（コンフリクトが起きやすいので注意）

**git status**<br>
ローカルリポジトリの状態の確認

**mkdir dirName**<br>
ディレクトリを作成する

# gitについて

## 主に使用するコマンド

**git clone**
リモートリポジトリのデータをローカル環境に持ってくる

**git add .**
編集内容を追加

**git commit -m"coment"**
addで追加したものを保存する

**git push URL**
commitでローカルに保存したものをリモートリポジトリに保存する

**git pull URL**
mergeとフォルスを同時に行う（コンフリクトが起きやすいので注意）

**git status**
ローカルリポジトリの状態の確認

**mkdir dirName**
ディレクトリを作成する
